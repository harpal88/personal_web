Projects
Project 1: Scanned PDF Data Extraction Engine
Developed a Python engine for extracting data from scanned PDFs, overcoming challenges like zoom, angle, noise, and variations. Utilized JSON mappings for field detection and stored extracted data in an SQL database. Implemented secure API endpoints for data upload and retrieval.
Project 2: Analytics Dashboard
Developed a comprehensive analytics dashboard using React.js, designed to provide real-time data visualization and insights. The dashboard integrates multiple data sources, offering customizable widgets and interactive charts. It features user-friendly navigation, filtering options, and secure access controls, enabling users to monitor key metrics and trends effectively.
Project 3: Custom Video Player
Designed and developed a custom video player application, leveraging HTML5, CSS3, and JavaScript. The player supports a range of video formats and includes features such as playback controls, fullscreen mode, customizable subtitles, and responsive design. Additionally, it integrates with a backend API for content streaming, providing a seamless and engaging viewing experience across devices.
